Browsing Object Stores

The main view of the ECM Developer plug-in is the Object Stores View. You can open this view using the Window > Show view >Other command. The Object Stores view is located in the Content Engine category. Selecting this view will show the following window:

If this window is empty you have to setup your connections to the Content Engine. See the getting started page for details.

First you have to make a connection to the Object Store using either the context menu of the Object Store or the toolbar button () of the view. When the Object Store is connected the Object Store can be browsed, showing folders, documents and custom objects. Also additional Object Stores can be added to this view using the toolbar button (). Next the operations currently possible on Object Store items will be discussed.

Refresh

An Object Store item can be refreshed using either the File > Refresh command or the Refresh command of the context menu. This command will get a fresh copy of the item on the server.

Rename

An Object Store item can be renamed using either the File > Rename command or the Rename command of the context menu. A dialog will appear where a new name for the item can be entered:

Move

An Object Store item can be renamed using either the File > Move command or the Move command of the context menu. A dialog will appear where a new location for the item can be entered:

In case of a multiple selection the new location will apply for all the items selected.

Delete

An Object Store item can be deleted using either the Edit > Delete command or the Delete command of the context menu. A confirmation is request for each item to be deleted.
